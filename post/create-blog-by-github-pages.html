如何通过github pages创建博客?

从很久前就一直想搭建个人博客，自己也写过好几个不同的博客系统。但是奈何每年的服务器费用是一笔不小的开支，自己也早已不是各大云厂的新用户，所以这个事情就一拖再拖。

直到最近通过创建github pages才得以实现。这篇文章就给大家来讲一讲如何通过github来创建个人博客!

创建github pages的前提是需要已经注册了github的账号，这里我就不讲解如何注册github的账号，以及github现在无法访问的问题，有这些疑问的读者请自行百度。

1、创建github仓库

首先是需要创建一个github仓库，仓库的类型为Public, 仓库的名称需要设置为账号.github.io否则创建后需要在域名后加上仓库的名称才可以访问，并且账号.github.io这个域名就作为我们博客的访问域名。

创建github仓库

创建完仓库以后，根据对应的提示，先初始化一下仓库。

2、进入设置页面

创建了仓库以后，我们进入到Settings页面，对仓库进行github pages配置。

点击进入设置页面

紧接着在Settings页面找到 GitHub Pages选项。然后分别设置对应的分支，主题，以及自定义域名。

找到github pages选项设置

在设置了自定义的域名以后，还需要进行域名解析，将当前的访问域名CNAME到自定义域名上。如果不设置自定义域名可直接通过账号.github.io进行访问。
